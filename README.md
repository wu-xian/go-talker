# go-talker

一个简易的聊天室,使用Go语言开发。

## 引用包
- `github.com\astaxie\beego\logs` 日志
- `github.com\cjbassi\termui` 终端渲染
- `github.com\golang\protobuf` 消息数据序列化和反序列化工具
- `github.com\urfave\cli` 服务端CLI
